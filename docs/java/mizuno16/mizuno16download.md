# 마인크래프트 Mizuno's 16 Craft 리소스팩 최신 버전 다운로드 방법

## 무슨 문제가 있었는가
이유는 아무도 모르지만 Mizuno's 16 Craft의 [공식 홈페이지](https://mizunomcmemo.blogspot.com/) 다운로드 방법이 페이지가 내려가서 404가 뜬다.

## 다운로드 

- [원본 파일 Medifire 링크](https://www.mediafire.com/file/42pqrtiwkmali5j/Mizuno's_16_Craft_JE_1.20.4-1.0_230105.zip/file)

재업로드는 아니고 페이지가 내려가기 전 사이트에 있던 Medifire 링크다. 여기서 먼저 다운로드를 받고,

- Modern Mizuno's
   - [Modrinth 다운로드](https://modrinth.com/resourcepack/modern-mizunos)
   - [CurseForge 다운로드](https://www.curseforge.com/minecraft/texture-packs/modern-mizunos)

이건 Mizuno's 16 Craft에서 기존 스타일을 최대한 따라해 최신 버전 블록들의 텍스쳐를 추가하는 리소스팩이다. 크게 이질감이 들진 않으니 최신 버전에선 이걸 같이 써주는것이 좋다.
